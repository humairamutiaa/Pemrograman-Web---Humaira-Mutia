Kesimpulan Job Sheet 1
Latihan 1.1 1. Spasi tanpa 'nbsp' akan diabaikan oleh browser.
2. '& nbsp' digunakan untuk membuat spasi panjang.
3. 'align' untuk mengatur posisi teks.
Latihan 1.2 1. Memberitahu bagaimana cara untuk membuat warna latar belakang.
2. Memberitahu bagaimana cara untuk membuat heading
3. Tag heading hanya untuk membuat judul, bukan untuk teks tebal.
Latihan 1.3 1. Menjelaskan tentang ukuran panjang-lebar dan warna pada garis.
2. 'color' digunakan untuk menetukan warna garis.
3. 'width' digunakan untuk mengatur lebar garis.
4. 'size' digunakan untuk menentukan ukuran garis.
5. Memberitahu bagaiman cara membuat komentar yang
tidak akan ditampilkan pada layar browser.
Latihan 1.4 1. Menjelaskan bagaimana cara mengubah berbagai macam teks.
2. < b > dan < strong > untuk membuat teks yang menjadi tebal.
3. < big > untuk membuat teks menjadi lebih besar.
4. < em > untuk membuat teks menjadi miring.
5. < small > untuk membuat teks lebih kecil.
6. < sub > teks ditampilkan lebih kecil dan di bawah garis dasar.
7. < sup > teks ditampilkan lebih kecil dan di atas garis dasar.
Latihan 1.5 1. < pre > untuk menandai bagian teks yang akan
ditampilkan secara preformat dan cocok untuk menampilkan kode computer.
2. Teks yang di-preformat akan menampilkan spasi dan line break apa adanya.
Latihan 1.6 1. < code >, < kbd >, < tt >, < samp >, dan < var > digunakan untuk
menunjukkan teks tertentu, seperti kode komputer, masukan keyboard,
teks teletype, contoh output, dan variabel.
2. < address > digunakan untuk menampilkan informasi alamat.
3. < abbr > dan < acronym > digunakan untuk menampilkan singkatan atau akronim.
Latihan 1.7 1. Menjelaskan bagaimana cara menandai kutipan teks.
2. < bdo > untuk mengatur arah penulisan teks dari kanan ke kiri.
3. < blockquote > untuk menandai blok teks yang merupakan kutipan panjang.
4. < q > untuk menandai kutipan pendek
Latihan 1.8 1. < del > untuk menandai teks yang dihapus dan memberikan garis pada teks yang dihapus.
2. < ins > untuk menandai teks yang disisipkan dan memberikan garis bawah pada teks yang disisipkan.
Latihan 1.9 1. Menjelaskan bagaimana cara membuat list.
2. < ul > untuk mulai membuat sebuah daftar tanpa urutan (unordered list).
3. < li > unutk menandai setiap item dalam list.
Latihan 1.10 1. Menjelaskan bagaimana cara membuat list bernomor.
2. < ol > untuk mulai membuat sebuah list dengan urutan (ordered list).
3. < ol type > unutk menentukan jenis penomoran list.
Latihan 1.11 1. Menjelaskan berbagai tipe bullet points, seperti disc, lingkaran, dan kotak.
2. "disc" untuk membuat bullet berbentuk disc.
3. "circle" untuk membuat bullet berbentuk lingkaran.
4. "square" untuk membuat bullet berbentuk persegi/kotak.
Latihan 1.12 1. Menjelaskan bagaimana cara membuat list bersarang dan list definisi dari istilah dalam daftar.
2. < dt > untuk menandai istilah.
3. < dd > untuk memberikan definisi dari istilah tersebut.
